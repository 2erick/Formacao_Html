Tags aprendidas

H1 ao H6

As heading tags fornecem uma estrutura hierárquica ao conteúdo, permitindo que os usuários identifiquem facilmente as seções principais e subseções de uma página. Elas ajudam a organizar o conteúdo em uma ordem lógica, facilitando a leitura e compreensão. Os motores de busca utilizam as heading tags para entender a estrutura e a relevância do conteúdo de uma página. O uso adequado das heading tags, com palavras-chave relevantes, pode ajudar a melhorar o ranqueamento nos resultados de pesquisa. Ao utilizar as heading tags de forma consistente e lógica, você facilita a leitura do conteúdo pelos visitantes da página. Cabeçalhos claros e bem organizados tornam o texto mais escaneável e ajudam os usuários a encontrar as informações relevantes de forma rápida.

O uso da tag H1 consiste em utilizá-la no título mais importante de sua página.

Eu sou uma tag h1

As heading tags seguintes (h2, h3, h4, h5 e h6) complementam o assunto principal.

Tag < p >

Representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco. Subdividir um conteúdo em parágrafos torna um texto mais acessível. Leitores de tela e outras tecnologias assistivas providenciam atalhos que permitem a navegação entre parágrafos. Possibilitando, então, uma leitura rápida do texto.

Tag < mark >

Representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular. Por Exemplo, pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque. No texto principal, o trecho destacado normalmente marca um conteúdo que pode ser de especial relevância para usuário corrente, como resultado de pesquisa.

Tag < small >

É usada para indicar que o texto dentro dela deve ser renderizado em um tamanho menor. É comumente usado para indicar texto legal, direitos autorais, notas de rodapé ou qualquer outra informação que seja de menor importância ou que precise ser visualmente distinta do restante do conteúdo.

I am small tag

Tag < i >

É usada para indicar que o texto dentro dela deve ser renderizado em itálico. Originalmente, era usada para indicar ênfase no texto, mas seu significado semântico mudou ao longo do tempo. Atualmente, é mais comumente usada para alterar a aparência do texto para itálico por razões estilísticas, sem implicar nenhum significado especial.

Este é um texto em itálico.

Tag < u >

A tag < u > é usada em HTML para envolver texto que deve ser exibido sublinhado no navegador da web. Quando você envolve texto com , o navegador renderiza esse texto com uma linha sob ele, indicando visualmente que é sublinhado. No entanto, é importante notar que o uso da tag não é considerado uma prática recomendada em HTML. Isso ocorre porque o sublinhado tradicionalmente indica um link clicável em muitos navegadores da web, e o uso de < u > para adicionar sublinhado a texto que não é um link pode causar confusão para os usuários.

Tag < strong >

A tag é usada em HTML para envolver texto que deve ser exibido como importante ou enfatizado. Ela geralmente é interpretada pelos navegadores como texto que deve ser exibido em negrito. Ao usar a tag , você está indicando ao navegador e aos leitores de tela que o texto contido dentro dela é particularmente significativo em relação ao restante do conteúdo. Isso pode ajudar a destacar palavras-chave, frases importantes, títulos, e outras informações cruciais no contexto da página.

Tag < ol >

A tag < ol > é usada em HTML para criar listas ordenadas, ou seja, listas nas quais os itens são enumerados com números ou letras. A estrutura básica de uma lista ordenada < ol > é composta por uma ou mais tags < li > (item da lista) dentro da tag < ol >. Cada < li > representa um item na lista.

Exemplo :

  1. Item 1
  2. Item 2
  3. Item 3

Tag < ul >

A tag < ul > é usada em HTML para criar listas não ordenadas, ou seja, listas nas quais os itens não são enumerados com números ou letras. A estrutura básica de uma lista não ordenada < ul > é composta por uma ou mais tags < li > (item da lista) dentro da tag < ul >. Cada < li > representa um item na lista.

Exemplo:

  • Item 1
  • Item 2
  • Item 3

Tag < a >

A tag < a > é usada em HTML para criar links, permitindo que os usuários naveguem entre diferentes páginas da web ou para outros recursos online, como arquivos, imagens, vídeos, etc. A tag < a > é um elemento de âncora e é uma das partes fundamentais da web para a criação de hipertexto. A tag < a > pode ser usada de várias maneiras, mas a forma mais comum é fornecer um texto ou imagem clicável que, quando clicado, leva o usuário para outro destino. Aqui está um exemplo simples: Visite nosso site

Tag < hr >

A tag < hr > em HTML é usada para criar uma linha horizontal, que é frequentemente usada para separar conteúdos ou seções em uma página da web. "HR" é uma abreviação de "horizontal rule" (regra horizontal).
A tag < hr > cria uma linha que se estende horizontalmente ao longo da largura do contêiner em que está contida. Por padrão, ela geralmente aparece como uma linha horizontal sólida.

Tag < sub >

A tag < sub > em HTML é usada para criar texto subscrito, o que significa que o texto dentro dessa tag é exibido ligeiramente abaixo da linha de base normal. É comumente usado para representar índices, como em fórmulas químicas (por exemplo, H2O) ou expressões matemáticas.

Tag < sup >

A tag < sup > em HTML é usada para criar texto sobrescrito, o que significa que o texto dentro dessa tag é exibido ligeiramente acima da linha de base normal. É comumente usado para representar expoentes, como em expressões matemáticas ou datas ordinais.(Por exemplo, 23).

Tag < blockquote >

A tag < blockquote > em HTML é usada para indicar que o texto dentro dela é uma citação longa ou um bloco de texto citado de outra fonte. Geralmente, os navegadores aplicam estilos específicos para este elemento, como recuo nas margens laterais, para destacar visualmente a citação.
Aqui está um exemplo de como a tag < blockquote > pode ser usada em HTML:

This is a long quotation that spans multiple lines. It may contain paragraphs, line breaks, and other text formatting.

Quoted text can come from various sources, such as books, articles, or speeches.

Tag < abbr >

A tag < abbr > é usada para representar uma abreviação ou acrônimo em um documento HTML. Ela pode incluir um atributo title para fornecer uma expansão ou descrição completa da abreviação quando o usuário passa o mouse sobre ela. Por exemplo:

O HTML é uma linguagem de marcação usada para criar conteúdo na web.
